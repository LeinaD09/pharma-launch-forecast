pharma-launch-forecast helps users understand and predict pharmaceutical launches in the German market. It offers models to forecast how generic drugs like Eliquis enter the market and how new brands compete, as seen in the GLP-1 class with Mounjaro vs. Ozempic.

This tool uses simple dashboards built with Streamlit. You can interact with the data, explore different scenarios, and even export results to Excel files. Its main goal is to make complex market forecasts accessible without needing any technical skills or coding.
